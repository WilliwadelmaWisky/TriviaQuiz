# Trivia Quiz
![License](https://img.shields.io/badge/license-MIT-green.svg?style=flat)

<p align="justify">
    A web application for taking multiple-choice quizzes. The quiz questions are fetched from OpenTriviaAPI. The application frontend is built in plain javascript without any external libraries and the application does not need a local server to work properly. Therefore using the application is extremely simple.
</p>

## Getting Started
### Installation
Clone the repository.

```
git clone https://github.com/WilliwadelmaWisky/TriviaQuiz.git
```

Open the `index.html` in a browser.
